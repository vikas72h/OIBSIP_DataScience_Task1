# Task 1 Iris Flower Classification

##  Overview

This project focuses on classifying Iris flower species using Machine Learning. The model predicts whether a flower belongs to Setosa, Versicolor, or Virginica based on sepal and petal measurements.

##  Objective

To build a classification model that accurately identifies Iris flower species using machine learning techniques.

##  Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

##  Project Workflow

* Data Loading & Exploration
* Data Visualization
* Feature Selection
* Model Training
* Prediction & Evaluation

##  Machine Learning Model

* Random Forest Classifier

##  Results

The model achieved high accuracy in classifying Iris flower species and was evaluated using Accuracy Score and Classification Report.

# Task 2: Unemployment Analysis with Python

## Overview

This project analyzes unemployment trends in India using Python. The dataset contains unemployment statistics from different states and regions of India. Through data cleaning, visualization, and exploratory data analysis, the project identifies unemployment patterns and examines the impact of the COVID-19 pandemic on employment across the country.

## Objective

The main objective of this project is to analyze unemployment rates in India, compare regional trends, and understand how COVID-19 affected employment opportunities and labour participation.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Project Workflow

* Data Loading and Understanding
* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Unemployment Rate Distribution Analysis
* Urban vs Rural Unemployment Comparison
* State-wise Unemployment Analysis
* Monthly Unemployment Trend Analysis
* COVID-19 Impact Analysis
* Correlation Analysis and Visualization

## Key Analysis Performed

* Distribution of unemployment rates across India
* Comparison of unemployment in urban and rural areas
* Identification of states with the highest unemployment rates
* Analysis of monthly unemployment trends
* Study of COVID-19's impact on unemployment
* Correlation between unemployment, employment, and labour participation rate

## Results

The analysis revealed significant differences in unemployment rates across states and regions. A noticeable rise in unemployment was observed during the COVID-19 pandemic, indicating its strong impact on the Indian labour market. The visualizations helped identify regional trends, employment patterns, and the relationship between labour participation and unemployment rates.
